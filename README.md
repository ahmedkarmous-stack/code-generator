# Générateur de Code QR

Un site web moderne pour générer des codes QR avec une explication technique détaillée du processus de génération.

## Fonctionnalités

- Génération de codes QR en temps réel
- Téléchargement en PNG haute qualité
- Code QR auto-référencé du site
- Explication technique complète du fonctionnement des QR codes

## Technologies

- HTML5
- CSS3 (design moderne avec thème sombre)
- JavaScript vanilla
- QRCode.js (bibliothèque de génération)
